This is where I give credit to the people and technology that make this website possible.

This website is essentially my personal webspace so it figures I'm the primary (and only) maintainer. The design for the site is heavily based on Whompy's Libra design found at OSWD. I just simplified it a bit, and modified it so that it conforms to W3C recommendations and other standards. For this, I used the W3C HTML Validation Service. The code itself was worked on in GNU Emacs and other text editors.

My caricature was drawn by a talented artist at FaceZoo. I just modified the color scheme to fit this site. The smiley images were obtained from an install of phpBB.

The pages themselves are served by a Sun machine (running Apache on Solaris 8) at the University of Michigan. The CGI script for page hit counts is provided by ITCS. They have also been (extensively) tested on Galeon and Microsoft Internet Explorer. The blog is powered by b2.

My resume was typeset in LaTeX. For this, I used a resume class file, created and shared by Miklos Csuros. It was then converted to the ubiquitous Adobe pdf format with dvi to pdf conversion tools.

All the work on the site is licensed under a Creative Commons License.

That's as far as I can remember for now. If I've missed something or someone else's contribution, please holler, and I'll fix it.
