Browser and Platform Compatibility Notes

This website has been tested on Internet Explorer on MS WindowsME and MS WindowsXP. On the same platforms, it has also been tested on Firebird, a gecko embedded browser. On GNU/Linux, it has been extensively tested on Galeon, another gecko embedded browser. On UNIX, it has been tested on Netscape Navigator 6+. On MacOS(9 and X), it has been tested on Internet Explorer, Omniweb and Chimera. Since this website conforms to all major standards, there is no reason it shouldn't work on your browser. If you are currently on a platform or browser that's not on this list and it works, please let me know. Also, since Netscape Navigator 4.x, and below, have non standard css support, they are currently the only known set of browsers in which the site fails to render. Not only does it fail to render, it locks up the browser. So be very afraid. I suppose a similar thing should happen to an older version of Internet Explorer without css support, but I don't have a copy to test it with.
